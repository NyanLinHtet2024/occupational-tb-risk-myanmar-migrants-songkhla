# occupational-tb-risk-myanmar-migrants-songkhla
R scripts for epidemiological analysis of occupational and environmental risk factors for tuberculosis among Myanmar migrant factory workers in Songkhla Province, Thailand. Focus on workplace ventilation, crowding, dust exposure, and TB exposure among co-workers.
The data used in this study contain sensitive information from migrant
workers and are not publicly available due to ethical and confidentiality
restrictions.
